# User Guide

Your screen has on-screen controls that appear when you move your cursor to the edge. These are managed by SDG-WAYSHELL and configured by this package.

| Control | How to trigger | What it does |
|---------|---------------|--------------|
| Volume OSD | Move cursor to the **right edge** | Shows and adjusts audio volume with scroll and click |
| Brightness OSD | Move cursor to the **left edge** | Shows and adjusts screen brightness |
| Screenshot toolbar | Move cursor to the **top center** | Captures screenshots; switch between save to disk, clipboard, or editor |
| Elevated process bar | When running sudo/root processes | Shows elevated processes across monitors |
| Focused process bar | When switching windows | Shows the active window's resource usage |
| Monocle switchers | When using monocle or deck layout | Shows per-monitor window lists for quick switching |

These controls appear automatically when triggered and hide after a short pause. No configuration is needed for normal use.
